# Tea Minigames

Tea Minigames is a reusable Minecraft minigames engine and a first game built on top of it, targeting Spigot 1.20.5 or newer. Individual game plugins may require a higher minimum — see each game's own docs.

There are three pieces - pick the section that matches what you're doing:

- **Tea** - the engine. TeaCore provides arena lifecycle, maps, teams, kits, abilities, cooldowns, effects, i18n, theme, chat routing, stats, cross-server coordination, and (when enabled) the lobby UI: connection signs, `/play`, and lobby spawn / gamemode auto-management. Install TeaCore on any server that runs Tea games.
- **Ascend** - the first game built on Tea. A free-for-all evolution minigame where players climb a ladder of mob morphs.

New games built on Tea each live in their own plugin jar and get their own section in these docs.
